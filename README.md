## Tech stack

- Frontend
  - React
  - Typescript
  - Webpack
  - Tailwind
- Linters
  - Eslint
  - Stylelint

## Installation and local development

1. install [Nodejs](https://nodejs.org/)
2. Run `npm i`
3. Run `npm run start`

## Deployment

1. Repeat steps **1** and **2** from local development.
2. Run `npm run build`
